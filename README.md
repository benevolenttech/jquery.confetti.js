# jquery.confetti.js
A jquery plugin to show confetti.

JsFiddle inspiration: http://jsfiddle.net/Javalsu/vxP5q/743/embedded/result/

Start:
------

    $.confetti.start();

or

    <button id="startConfetti"></button>

Stop:
-----

    $.confetti.stop();

or

    <button id="stopConfetti"></button>

Restart:
--------

    $.confetti.restart();

or

    <button id="restartConfetti"></button>
